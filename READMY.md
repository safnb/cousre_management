## Быстрый запуск

1. Перейдите в папку проекта:
   ```bash
   cd trade-app
   ```
2. Запустите контейнеры приложения и базы данных:
   ```bash
   docker compose up --build
   ```
3. API будет доступен по адресу:
   `http://localhost:8080`

4. Чтобы остановить контейнеры:
   ```bash
   docker compose down
   ```

## Документация API

После запуска через Docker Swagger-документация доступна по адресу:
`http://localhost:8080/api-docs`

# Управление курсами обучения

## Что делает проект

Система управления курсами обучения — это веб-приложение для создания и управления образовательными курсами. Система позволяет преподавателям публиковать материалы, назначать задания, отслеживать прогресс студентов, а студентам — удобно получать доступ к учебным ресурсам и взаимодействовать с материалом.

## Какие проблемы он решает

### Для преподавателей:
- **Временные затраты на администрирование** — автоматизирует проверку заданий, формирование отчетов, коммуникацию
- **Сложность отслеживания прогресса** — визуализирует успеваемость каждого студента и группы в целом
- **Ограниченные инструменты взаимодействия** — предоставляет встроенные средства для обсуждений, обратной связи и совместной работы

### Для студентов:
- **Неорганизованность материалов** — структурирует учебный контент по модулям и темам
- **Сложность отслеживания дедлайнов** — автоматически напоминает о сроках сдачи заданий
- **Отсутствие персональной траектории** — позволяет видеть свой прогресс и рекомендации для улучшения результатов

## Основные возможности

### 🎯 Управление курсами
- **Создание и структурирование курсов** по модулям и урокам
- **Гибкая настройка параметров**: длительность, уровень сложности, категория
- **Шаблоны курсов** для быстрого создания новых программ

### 👥 Управление пользователями
- **Ролевая модель**: студенты, преподаватели, администраторы, модераторы
- **Группы и потоки** для организации студентов
